PluginDetect Java Example # 3

In this example we force Java applets to run in the web page by setting verifyTags = [3, 3, 3].
Then we query the Java applets directly to get the plugin detection results.

For Internet Explorer, we try to run the Java applets using an <applet> tag, AND an HTML5 compatible <object> tag, AND an <object> tag with Oracle's Java ActiveX classid. For all other browsers, we try to run the Java applets using an <applet> tag AND an HTML5 compatible <object> tag.
